# Uncertainty-Aware Prediction of Thermal Expansion in Complex Oxides

This repository contains the dataset and code used in the manuscript:

**“Uncertainty-Aware Prediction of Thermal Expansion in Complex Oxides via LLM-Curated Literature Data”**

## Contents
- `Final_CTE_Code_Main.ipynb` — notebook used for model development, training, and evaluation
- `cte_dataset.xlsx` — curated dataset of experimentally reported coefficients of thermal expansion
- `figures/` — manuscript figures and supplementary figures, if provided

## Workflow
The repository supports:
1. literature-curated CTE dataset construction
2. composition preprocessing and normalization
3. feature generation
4. probabilistic machine learning for CTE prediction
5. uncertainty calibration and evaluation

## Reproducibility
All data and code required to reproduce the reported results are provided in this repository.
